In this folder, there are three library files (libKDCReader.a) built for different purposes.

1. .\libKDCReader.a
   - built for development purpose
   - support both iOS devices and iOS Simulators

2. .\Release-iphoneos\libKDCReader.a
   - built for release purpose
   - support only iOS devices

3. .\Release-iphonesimulator\libKDCReader.a
   - built for development purpose
   - support only iOS Simulators
